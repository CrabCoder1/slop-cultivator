# Game Keybinds

This document tracks all keyboard shortcuts and keybinds used in Slop Cultivator.

## Gameplay Controls

### Range Indicator Visibility
- **Alt (Hold)**: Show attack range indicators for all deployed cultivators
  - Range indicators are also shown when:
    - A new cultivator is selected for deployment (placement preview)
    - Hovering over a deployed cultivator
    - A cultivator is selected

### Health Bar Visibility
- **Alt (Hold)**: Show health bars for all damaged cultivators
  - Health bars are also shown when:
    - Hovering over a damaged cultivator
    - A damaged cultivator is selected
  - Health bars only appear when a cultivator has taken damage (health < 100%)

### Placement Controls
- **Right Click**: Cancel cultivator placement mode
- **Left Click**: Place cultivator (when in placement mode and position is valid)
